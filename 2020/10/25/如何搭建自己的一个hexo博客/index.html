一、GitHub注册

一、GitHub学生包申请

1、为什么要申请GitHub学生包
Github 学生包包含很多开发者工具，比较吸引的是 DigitalOcean 的 50美元 代金券和 NameCheap 的免费 .me 域名和 Comodo 的 SSL 证书
2、申请条件
年龄大于 13 岁并正在进行可能被授予学位或学历的课程，意味着你从初中开始就可以申请学生包了。你只需要一个学校提供的电子邮箱地址，或有效的学生证件或是任何官方入学证明。
3、申请教程
这里不多阐述，可以在网上随意的找到，附上一个csdn里的教程：GitHub学生包申请

二、GitHubc创建一个仓库

1、打开GitHub主页，选择start a project

2、填写仓库信息，填好后点击create repository
仓库的名字要 你注册时的名字.github.io

二、Git

一、Git的安装

1、获取git的安装包

1、直接从官网下载（比较慢）：Git

2、在百度搜索git的下载安装，选择他们分享的百度网盘链接

2、Git的安装

双击下载好的git安装程序，开始安装，Windows上的git安装没什么特别要调的东西，除了安装位置稍作修改以外，可以一直按Next（当然你想装在C盘那就全程Next），最后Install安装
